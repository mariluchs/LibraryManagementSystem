# Event Manager Projekt zur LV Programmierung industrieller Informationssysteme mit Java EE 

## Branches

### main
Enthält den aktuellen Stand.

Reihenfolge der Branches:

0. ``start`` Projektsetup
1. ``komponentenweb`` Komponentenbasierte Web-Benutzerschnittstellen
2. ``jpa`` Datenbankanbindung
3. ``komponentenbibiliothek`` Implementierung von Formularen etc
4. ``user_management`` Login, Registrierung
5. ``user_access_management`` Einfaches UAM ohne Jakarta EE
6. ``role_management`` Implementiert verschiedene User-Rollen
7. ```user_role_annotation``` Implementiert @OrganizerOnly annotation
8. ```business_logic``` Einfache Geschäftslogik
9. ```manytomany``` Setzt die M2M Relation als Join Entity Table um
10. ```myevents``` Implementiert eine Seite, die einem Teilnehmer zeigt, zu welchen Events er sich angemeldet hat
